## Что такое маркетплейс

Маркетплейс — это онлайн-платформа, которая соединяет продавцов и покупателей в одном месте. В отличие от классического интернет-магазина, маркетплейс не продаёт собственные товары, а предоставляет инфраструктуру: витрину, платёжную систему, логистику и аудиторию.

> **Простая аналогия:** если интернет-магазин — это отдельный ларёк на улице, то маркетплейс — это торговый центр, куда люди уже приходят с намерением купить.

### Ключевые характеристики маркетплейса

| Характеристика | Описание |
|---|---|
| **Модель** | Площадка-посредник между продавцом и покупателем |
| **Монетизация** | Комиссия с продаж, платное продвижение, логистические услуги |
| **Трафик** | Обеспечивается платформой |
| **Логистика** | Может быть на стороне площадки (FBO) или продавца (FBS) |
| **Клиентская база** | Принадлежит маркетплейсу, а не продавцу |

## Как работают маркетплейсы

### Модели работы

```
┌──────────────────────────────────────────────────┐
│                  МАРКЕТПЛЕЙС                     │
│                                                  │
│   Продавец ──► Склад МП ──► Покупатель   (FBO)  │
│   Продавец ──────────────► Покупатель     (FBS)  │
│   Продавец ──► Склад МП ──► ПВЗ ──► Покупатель  │
└──────────────────────────────────────────────────┘
```

- **FBO (Fulfillment by Operator)** — товар хранится на складе маркетплейса. Площадка сама упаковывает и доставляет. Продавец платит за хранение.
- **FBS (Fulfillment by Seller)** — товар хранится у продавца. При заказе продавец сам отправляет товар в сортировочный центр или напрямую покупателю.
- **DBS (Delivery by Seller)** — продавец полностью контролирует доставку. Маркетплейс выступает только витриной.

### Схема комиссий

```
Цена товара для покупателя
│
├── Себестоимость товара
├── Комиссия маркетплейса (5–25%)
├── Логистика (доставка до покупателя)
├── Хранение на складе (если FBO)
├── Эквайринг (~1–2%)
├── Возвраты и брак (~3–10%)
└── Прибыль продавца (то, что осталось)
```

## Преимущества и недостатки

### ✅ Преимущества

- **Готовый трафик.** Миллионы покупателей уже на площадке. Не нужно тратить месяцы на SEO и рекламу.
- **Низкий порог входа.** Зарегистрировался — начал продавать. Без аренды, без сотрудников.
- **Инфраструктура.** Склады, доставка, возвраты, эквайринг — всё берёт на себя площадка.
- **Доверие покупателей.** Люди охотнее покупают на известной площадке, чем в незнакомом магазине.
- **Масштабирование.** Можно быстро выйти на большие объёмы продаж.

### ❌ Недостатки

- **Высокая конкуренция.** Сотни продавцов с аналогичным товаром.
- **Зависимость от площадки.** Правила меняются без предупреждения. Аккаунт могут заблокировать.
- **Нет своей клиентской базы.** Покупатель лоялен маркетплейсу, не вам.
- **Растущие комиссии.** Маржинальность снижается с каждым годом.
- **Ценовые войны.** Конкуренты демпингуют — приходится снижать цены.
- **Возвраты.** Особенно в fashion-сегменте возвраты могут достигать 30–50%.

## Как начать продавать

### Шаг 1. Выбор ниши и товара

Прежде чем выходить на маркетплейс, нужно понять, что продавать. Критерии хорошего товара:

- Стабильный спрос (проверяйте через аналитические сервисы)
- Маржинальность от 30% (иначе комиссии съедят прибыль)
- Небольшой вес и габариты (меньше расходы на логистику)
- Низкий процент возвратов
- Не слишком высокая конкуренция

### Шаг 2. Регистрация на площадке

```bash
# Что понадобится:
- ИП или ООО (самозанятые — только на некоторых площадках)
- Расчётный счёт
- Документы на товар (сертификаты, декларации)
- ЭЦП (для некоторых площадок)
```

### Шаг 3. Создание карточки товара

Карточка товара — ваш единственный «продавец». Она должна быть идеальной:

1. **Фото.** Минимум 5–7 качественных фото на белом фоне + инфографика.
2. **Заголовок.** Содержит ключевые слова, по которым ищут товар.
3. **Описание.** Структурированное, с ключевыми характеристиками.
4. **Rich-контент.** Видео, 360°-фото, A+ контент.
5. **Отзывы.** Критически важны — товар без отзывов почти не покупают.

### Шаг 4. Логистика

Определитесь с моделью:

| Модель | Когда подходит |
|---|---|
| FBO | Стабильный спрос, стандартные товары |
| FBS | Крупногабаритные товары, тестирование ниши |
| DBS | Хрупкие/дорогие товары, нужен контроль |

### Шаг 5. Запуск и продвижение

После первых продаж подключайте внутреннюю рекламу площадки и работайте над отзывами.

## Юнит-экономика

Перед выходом на маркетплейс посчитайте юнит-экономику — сколько вы зарабатываете (или теряете) на каждой единице товара.

### Формула

```
Прибыль = Цена продажи - Себестоимость - Комиссия МП - Логистика - Хранение - Возвраты - Реклама
```

### Пример расчёта

```
Цена продажи:          2 000 ₽
Себестоимость:          - 600 ₽
Комиссия WB (15%):     - 300 ₽
Логистика:              - 150 ₽
Хранение:               - 50 ₽
Возвраты (10%):         - 200 ₽
Реклама:                - 200 ₽
──────────────────────────────
Прибыль:                  500 ₽ (25% от цены)
```

> ⚠️ **Важно:** если маржинальность ниже 15–20%, любое изменение условий площадки может сделать бизнес убыточным.

## Продвижение на маркетплейсе

### Внутренние инструменты

- **Поисковая оптимизация (SEO).** Ключевые слова в заголовке и описании.
- **Внутренняя реклама.** Платное продвижение в поиске и каталоге.
- **Акции и скидки.** Участие в распродажах площадки.
- **Отзывы.** Мотивируйте покупателей оставлять отзывы с фото.

### Внешние инструменты

- **Социальные сети.** Трафик из Instagram, Telegram, VK.
- **Блогеры.** Обзоры товара у инфлюенсеров.
- **Контекстная реклама.** Яндекс Директ с переходом на карточку.
- **Telegram-канал.** Собственная аудитория вне маркетплейса.

### Воронка продвижения

```
Внешний трафик ──► Карточка товара ──► Покупка ──► Отзыв
       │                                            │
       │                                            ▼
       │                                   Повторная покупка
       │                                            │
       ▼                                            ▼
  Узнаваемость бренда ◄───────────────── Лояльность
```

## Частые ошибки

### 1. Выход без расчёта юнит-экономики

> «Сначала начнём продавать, потом посчитаем» — путь к убыткам.

### 2. Плохие фотографии

Покупатель не может потрогать товар. Фото — единственный способ «продать» визуально. Экономия на фото = потеря продаж.

### 3. Одна площадка

Зависимость от одного маркетплейса — риск. Диверсифицируйте каналы продаж.

### 4. Игнорирование отзывов

Негативные отзывы без ответа убивают конверсию. Отвечайте на каждый отзыв, решайте проблемы.

### 5. Демпинг

Снижение цены ради объёма продаж без учёта маржинальности — это не стратегия, а самоубийство.

### 6. Отсутствие бренда

Без бренда вы — один из тысячи продавцов «такого же» товара. Бренд = возможность продавать дороже.

## Заключение

Маркетплейсы — мощный канал продаж, но не волшебная кнопка «деньги». Успех требует:

1. **Расчёта** — знайте свои цифры до выхода на площадку
2. **Качества** — карточка товара, фото, описание должны быть на высоте
3. **Диверсификации** — не зависьте от одной площадки
4. **Бренда** — стройте узнаваемость за пределами маркетплейса
5. **Адаптации** — правила меняются, нужно быстро реагировать
